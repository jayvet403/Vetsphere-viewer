Vetsphere Viewer - Embedding Examples

Example 1: Basic Embed

The simplest way to embed the viewer. This creates a full-sized viewer in your page.

<iframe
  src="https://jayvet403.github.io/Vetsphere-viewer/viewer.html"
  width="100%"
  height="800"
  frameborder="0">
</iframe>

Example 3: Load Study on Page Load

Pre-load a specific DICOM study by passing the studyInstanceUIDs parameter.

<iframe
  src="https://jayvet403.github.io/Vetsphere-viewer/viewer.html?studyInstanceUIDs=1.2.826.0.1.3680043.8.498.13391311735141438663280741492711973"
  width="100%"
  height="800"
  frameborder="0">
</iframe>
Example 4: Control Viewer with JavaScript

Use PostMessage API to communicate with the viewer from your parent page.

// Get reference to iframe
const frame = document.getElementById('viewer-iframe');

// Send message to load a study
frame.contentWindow.postMessage({
  type: 'LOAD_STUDY',
  studyInstanceUID: '1.2.826.0.1.3680043.8.498.13391311735141438663280741492711973'
}, '*');
Example 5: Responsive Container

Make the viewer responsive using CSS aspect-ratio and flexbox.

<style>
  .viewer-container {
    position: relative;
    width: 100%;
    aspect-ratio: 16 / 9;
  }
  .viewer-container iframe {
    position: absolute;
    top: 0; left: 0;
    width: 100%; height: 100%;
  }
</style>

<div class="viewer-container">
  <iframe src="https://jayvet403.github.io/Vetsphere-viewer/viewer.html" frameborder="0"></iframe>
</div>
